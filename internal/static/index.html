Como exportar seu Currículo Lattes em XML

Exportar o seu Currículo Lattes no formato XML é um processo rápido, mas a opção fica um pouco "escondida" nos ícones laterais. Esse formato é ideal para quem precisa fazer backups ou importar os dados em outros sistemas acadêmicos.

  1. Acesse a Plataforma Lattes: Vá ao site lattes.cnpq.br e clique em "Atualizar Currículo".
  2. Faça o Login: Entre com seu CPF e senha.
  3. Localize o ícone de Exportar: Na tela principal do seu currículo (onde você edita as informações), olhe para o menu de ícones no lado direito.
  4. Clique em Exportar: Procure pelo ícone que parece uma caixa com uma seta saindo para a direita (ou passe o mouse sobre os ícones até ler "Exportar").
  5. Escolha o Formato: Uma janela pop-up abrirá. Selecione a opção "Formato XML do Currículo Lattes". Clique em Confirmar.
  6. Download: O sistema vai gerar um arquivo compactado (geralmente em formato .zip). Salve-o no seu computador.
